# Projeto SCM 2025 — Janaína Cazuza

## Descrição

Sistema de Gestão de Configuração de Software (SCM) desenvolvido como atividade prática da disciplina de Projeto de Software, Unidade 4, do curso de Análise e Desenvolvimento de Sistemas.

O projeto simula o gerenciamento de itens de configuração de um repositório de software, permitindo o registro de artefatos, o controle de versões e a rastreabilidade das alterações realizadas ao longo do ciclo de desenvolvimento.

## Tecnologias utilizadas

- **Linguagem:** Java 17+
- **Controle de versão:** Git e GitHub
- **Padrão de commits:** Conventional Commits

```bash
javac SistemaGestaoConfiguracao.java
java SistemaGestaoConfiguracao
```
